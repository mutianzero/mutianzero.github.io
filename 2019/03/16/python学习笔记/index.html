在代码的开头要习惯加上：
! /usr/bin/python3
--coding:UTF-8--

如果能使用for循环，就不要使用while循环。

高级语言翻译成机器码有两种方法：解释和编译
解释就是边读代码边执行，解释器会每次读入一段代码，并执行它，接着再读入并执行，如此反复，直到执行结束。
python3中有6大标准的数据类型：Number、String、List、Tuple、Sets、Dictionary
python3支持3种不同的数值类型：int、float、complex
Variable(变量)
把一个变量a赋值给另一个变量b,就是把变量b指向变量a所指向的数据。
python3共有33个关键字。

列表：

python中的所有序列都可以进行一些特定的操作，包括索引indexing,分片slicing,序列相加adding,乘法multiplying,成员长度，资格，最大值和最小值。
number=[1,2,3,4,5,6,7,8,9]
number[0:3]表示从索引0到索引2的元素，即[1,2,3]
通过负的索引得到最后一个元素值：将第二个索引值设置为空就可以，即：num[-3:]就可以得到[7,8,9]
对于步长，默认为1，对于正数步长，从左向右提取元素，对于负数步长，从右向左提取元素，如果要间隔几个元素对元素进行提取，将步长设置为对应的数字就可以。
列表用[]表示，列表的元素是可变的。不能为一个不存在元素的位置赋值，一旦初始化列表后，就不能再往列表里增加元素了。append()函数可以在列表末尾添加新的元素，del可以通过索引的方式直接删除列表中的元素。
分片赋值的一个强大的功能：可以使用与原序列不等长的序列将分片替换，也可以在不替换任何原有元素的情况下在任意位置插入新元素。
列表可以嵌套，嵌套后还是列表，
extend()方法用于在列表末尾一次性追加另一个序列中的多个值（用新列表扩展原来的列表）。extend()方法修改了被扩展的序列，
append()方法用于在列表后追加单个值。
index()方法用于从列表中找出某个值第一个匹配项的索引位置，即参数是要查找的元素，返回值是该参数在列表中的索引值
语法：list.index(obj)
insert()方法用于将对象插入列表，语法如下：
list.insert(index,obj)
插入的位置是，index之前，index-1之后。
pop方法用于移除列表中的一个元素，（默认是最后一个元素）,并且返回该元素的值。

pop()方法是唯一一个既能修改列表又能返回元素值（除了None）的列表方法。
使用pop方法可以实现栈。栈里的元素是后进先出（LIFO），入栈push,出栈pop,python没有入栈方法，但是可以使用append代替，pop方法和append方法的操作结果相反，如果入栈刚刚出栈的值，最后得到的结果就不会变。
python列表的排序算法：sort()和sorted()
sorted返回一个新的list，不对原本的list进行修改。
sort将原本的list修改，不保留原本的list，
用法：sorted(list),list.sort()
sort()的语法：list.sort(func)，#func为可选参数。如果指定该参数就会使用该参数的方法进行排序。
如果需要对一个序列进行反向迭代，可以使用reversed()函数，该函数不返回list，而是返回一个迭代器（iterator）对象，可以通过List函数吧返回的对象转换为list,例：num=[1,2,3]
list(reversed(num))结果为：[3,2,1]
要想对一个列表进行排序同时还要保留原列表，则可以将原列表进行分片后赋值给另一个变量，如果不分片就进行赋值，那么原列表和赋值的列表指向的都是同一个列表，故进行该操作前，一定要分片。或者直接使用sorted()函数，sorted()函数保留原列表。该函数可以直接获取列表的副本进行排序，语法：sorted(list)
clear()方法用于清空列表，类似于del list[:],
语法：list.clear(),不需要传入参数，要谨慎使用。
copy()方法，用于复制列表。类似于list[:];
语法：list.copy(),不需要传入参数。是原原本本的拷贝操作。
高级排序：
如果需要元素能按照特定的方式进行排序（不是sort方法默认的按升序排序元素），就可以自定义该方法。
sort方法有两个可选的参数，reverse和key。若要使用就要通过名字指定，称之为关键字参数。
例如：按照字符串由短到长排序：

fie=[‘study’,’python’,’is’,’interesting’]
fie.sort(key=len)
fie
[‘is’, ‘study’, ‘python’, ‘interesting’]
按照字符串由长到短排序，需要两个参数，key=len,reverse=True
fie.sort(key=len,reverse=True)
fie
[‘interesting’, ‘python’, ‘study’, ‘is’]

元组tuple：

python的元组元素不可修改。
创建元组很简单，用逗号分隔了一些值，就会自动创建元组。元组使用圆括号（）,也可以创建空元组。()。
元组的函数：
tuple()函数可以创建一个元组，如果参数是元组，参数就会被原样返回。如果是以一个序列作为参数，就会将其转换为元组。
访问元组：
可以使用下标索引访问元组中的值，这一点和列表List一样。
修改元组：
元组的元素不可修改，但是可以对元组的元素进行连接结合，和list一样。
删除元组：
元组中的元素不可被删除，但是可以使用del语句删除整个元组。
语法：del tuple
删除后就不能再对元组进行操作，会提示异常，因为该元组已经不存在。
元组索引、截取
这一点和List一样，可以使用索引，对元组中的一段元素进行操作。
语法：tuple[n],tuple[1:]等。
内置函数：
len()计算元组元素个数
max(tuple),min(tuple),tuple(seq)将列表转换为元组。seq为list。
元组与列表的区别：
因为元组元素不可更改，因此其代码更安全，如果可能，在能用元组代替列表的情况下，尽量代替。
元组的元素不能改变的含义：每个元素的指向永远不变，指向一个list对象就不能改成指向其他对象，但指向的list列表本身是可以改变的。比如在一个元组里嵌套了一个list,该list的元素内容是可以改变的。
分片操作不存在索引越界的问题，分片操作对不存在的编号返回空值，存在的返回对应的值，例如：

number=[1,2,3,4,5,6]
len(number)
6
number[3:15]
[4, 5, 6]
number[-100:-1]
[1, 2, 3, 4, 5]
number[-100:]
[1, 2, 3, 4, 5, 6]
字符串：
所有标准序列操作（索引、分片、成员资格、求长度、去最大值、最小值等）对字符串同样适用，字符串是不可变的，所以对字符串无法分片赋值。
字符串格式化符号：%，%s
例：
print(‘hello,%s’%’world’)
hello,world
适用%f,若不指定精度，默认输出6位小数。
指定输出的小数位数：例：输出2位小数：%.2f,在python中，字符串中的%是转换说明符，如果要输出%，就需要格式化字符%，从而需要使用%%
对格式化字符串的说明：只有元组和字典可以格式化一个以上的值，如果使用列表或其他序列代替元组，序列就会被解释为一个值。
例：在一个句子中格式化了多个值，则占位符应该写成%(‘’,’’,..)
字符串方法：
find(),用于检测字符串中是否包含子字符串str。如果指定了beg(开始)和end(结束)范围，就检查是否包含在指定范围内，如果包含该子字符串，就返回开始的索引值，否则返回-1.
语法：str.find(str,beg=0,end=len(str))#str表示指定检索的字符串，find()方法返回的不是boolean值
join()方法：
用于将序列中的元素以指定的字符连接成一个新的字符。
语法：str.join(sequence)
例：
mark=’+’
fie=[‘1’,’2’,’3’]
mark.join(fie)
‘1+2+3’

str表示检索的字符串，sequence表示要连接的元素序列，返回结果为指定字符连接序列中元素后生成的新字符串。进行join操作调用和被调用的对象都必须都是字符串，任意一个不是字符串都会报错。
lower()方法：
将字符串中所有的大写字母转换成小写字母。语法：str.lower()
upper()方法：
用于将字符串中的小写字母转换层大写字母。语法：str.upper()
返回值为：转换后的大写字母。
swapcase()：
该方法用于将字符串中的大写字母转换为小写，小写字母转换成大写。
语法：str.swapcase()
replace()方法：
如果指定第3个参数Max,替换次数就不超过Max.
语法：str.replace(old,new[,max])
str表示指定检索的字符串，old表示将会被替换的子字符串，new表示新字符串，用于替换old子字符串，max代表可选字符串，替换次数不超过max次。
split()方法：
通过指定的分隔符对字符串进行切片，如果参数num有指定值，就只分隔num个子串，是join()的逆方法。语法：str.split(str=””,num=string.count(str)),str表示指定检索的字符串，st代表分隔符，默认为空格，num代表分割的次数，返回值Wie分割后的字符串列表。
例：

str=’do it now’
str.split()
[‘do’, ‘it’, ‘now’]
strip():
用于移除字符串头尾指定的字符，默认为空格。
语法：str.strip([chars])
chars表示要移除的字符。
例：
str=’__hello_‘
str.strip(“_”)
‘hello’
translate()方法：
该方法根据参数table给出的表(包含256个字符)转换字符串的字符，将要过滤掉的字符放到del参数中。
语法：
str.translate(table[,deletchars])
str表示指定检索的字符串，table代表翻译表，翻译表通过maketrans方法转换而来，deletchars代表字符串中要过滤掉的字符列表，返回结果为为翻译后的字符串。

字符串比较：可以按照字母的排序进行比较。
用and可以连接两个不同的布尔类型的值，比如：if num<5 and="" num="">1:
这一点不同于其他语言，需要用 &&运算符连接：if num<5&&num>1,与and同类型的还有or、not。在用这三个运算符比较时，会进行短路运算，即若用and运算符，当左右的布尔值为0时，立马会返回0，不会再与右边的进行比较。
断言assert：
其存在的意义：在没有完善一个程序之前，不知道程序哪里会出错，与其在运行时崩溃，不如在出现错误条件时就崩溃。assert关键字可以实现这种工作方式。当assert后面的条件为真时，程序正常运行，为假时，输出错误的信息。错误的提示信息可以自定义，这个错误的提示信息被称为：异常参数。assert的异常参数就是在断言表达式后面添加的字符串信息，用来解释断言并更容易知道问题出在哪里。
使用assert要注意：
1、assert断言是用来声明某个条件为真。
2、assert语句失败时，会引发一个AssertionError
字典中的元素是没有顺序的，也就是说，迭代时字典中的键和值都能保证被处理，但是处理的顺序不确定，这也是使用for循环输出字典中的元素时不按照顺序输出的原因。
跳出循环：break,continue
break和continue用在while和for语句中，如果使用嵌套循环，break会跳出最内层循环，当遇到continue语句时，无论执行条件是否为真，都跳过这次循环，进入下一次循环。相比break语句，使用continue只跳过一次循环，不会跳出整个循环。
pass语句：python中不能有空代码，当在某个语句中什么都不需要执行时，加上一个pass就行了。
在windows上不能像执行exe一样执行py文件，在Mac和Linux上可以，方法是：在文件的第一行加上：#!/usr/bin/env python3
python有两种循环,或者说三种，分别是：for,while,和for与while的循环嵌套，没有do while循环。

函数：
没有return语句时，相当于返回None
return None也可以简写为：return
return表达式表示结束函数，选择性返回一个值给调用方，不带表达式的return相当于返回None
python定义函数要用def关键字。
调用函数时可以使用以下参数类型：
1、必须参数，2、关键字参数，3、默认参数，4、可变参数，5、组合参数
定义函数时若定义了必须参数，在调用函数时，就必须传入定义的参数，否则出错。
关键字参数，允许调用函数时参数的顺序和声明时的顺序不一样，Python的解释器能够用参数名匹配参数值。
默认参数：在声明函数时，对参数设定了默认值，在调用时，若没有传入对应的参数，调用的函数就会使用这个默认值。
可变参数：
定义语法：
def function([formal_args,] var_args_tuple):
‘函数_文档字符串’
function_suite
return[expression]
加了号的变量名会存放所有未命名的变量参数。如果变量参数在函数调用时没有指定参数，就是一个空元组。也可以不向可变函数传递未命名的变量。
组合参数：
即对以上四种参数组合使用。
对于函数：
def persion(age,name):
print(‘年龄：’age)
print(‘名字：’name)
在函数体内都是对形参进行操作，不能操作实参，这个函数参数列表的参数是实参，而函数体内的参数是形参。

两种最基本变量的作用域：
局部变量和全局变量：
在函数体内部定义的变量是局部变量，函数执行完后，该局部变量被释放。

函数每次被调用时都会创建一个新的命名空间，也就是当函数调用自己时，实际上运行的是两个不同的函数（也可以说一个函数具有两个不同的命名空间）
使用递归函数需要注意防止栈溢出，在计算机中，函数调用是通过栈这种数据结构来实现的，每当进入一个函数调用时，栈就会加一层栈帧；每当函数返回，栈就会减少一层栈帧。由于栈的大小不是无限的，因此多次调用会导致栈溢出。
栈溢出的解决方法：通过尾递归优化解决。
尾递归和循环的效果一样。尾递归是指：函数返回时调用函数本身，并且return语句不能包含表达式
