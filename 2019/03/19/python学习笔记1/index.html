pytho的字符串不能被改变。比如：word=’python’,word[0]=’m’是错误的。
列表（list）中的元素是可以改变的，比如:a=[1,2,5,8,9],可以执行a[0]=8，列表是有序的。
Tuple（元组）：tuple=(‘ada’,15485,89,’robin’,70.2)，元组的元素不可修改，修改元组的元素是非法的。这一点同字符串一样。
string、list和tuple都属于sequence（序列）。
set（集合），是一个无序不重复元素的序列，（无序性，确定性，互异性）
字典是无序的对象集合，字典中的元素是通过键来存取的，而不是通过偏移存取，字典是一种映射类型，用大括号{}标识，在同一个字典中，键（key）必须是唯一的。再次强调，字典是无序的。
比如tinydict={‘name’:’robin’,’code’:’1’,’site’:’www.runoob.com'}
tiny（微小的）,关于字典的三个注意点：1、字典是一种映射类型，它的元素是键值对。2、字典的关键字必须为不可变类型，且不能重复。3、创建空字典使用 { }。

python中时间日期格式化符号：
%y 两位数的年份表示（00-99）
%Y 四位数的年份表示（000-9999）
%m 月份（01-12）
%d 月内中的一天（0-31）
%H 24小时制小时数（0-23）
%I 12小时制小时数（01-12）
%M 分钟数（00=59）
%S 秒（00-59）
%a 本地简化星期名称
%A 本地完整星期名称
%b 本地简化的月份名称
%B 本地完整的月份名称
%c 本地相应的日期表示和时间表示
%j 年内的一天（001-366）
%p 本地A.M.或P.M.的等价符
%U 一年中的星期数（00-53）星期天为星期的开始
%w 星期（0-6），星期天为星期的开始
%W 一年中的星期数（00-53）星期一为星期的开始
%x 本地相应的日期表示
%X 本地相应的时间表示
%Z 当前时区的名称
